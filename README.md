# 🎨 우리 가족, 행복한 순간 그리기 (v2.6 학습지 2페이지 최적화)

> 💖 가족과의 소중한 경험을 글과 그림으로 표현하는 교육용 인터랙티브 웹 활동지

## 🌟 주요 기능

### 📝 **활동 구성**
- **🟡 도입**: 가족과의 행복한 순간 떠올리기
- **🔵 활동1**: 엽서로 표현하기 (사진 업로드 + 자유 메시지 작성)
- **🔴 활동2**: 공감 이름표 만들기 (가족과의 소통 내용)
- **🟢 마무리**: 감정 정리하기 (이모지 선택 + 이유 작성)

### 🚀 **v2.6 핵심 개선사항 (NEW!)**
- 📄 **선생님용 학습지 2페이지 구조**: 내용 잘림 문제 완전 해결
- 📐 **A4 최적화 레이아웃**: 각 페이지별 내용량 균등 배분
- 🎯 **완벽한 인쇄 품질**: 210mm × 297mm A4 크기에 정확히 맞춤
- ✍️ **추가 작성 공간**: 2페이지에 "추가로 하고 싶은 말" 섹션 추가
- 🔄 **자동 페이지 분할**: JavaScript로 1페이지, 2페이지 각각 별도 생성

### 🎯 **v2.5 기존 기능 (유지)**
- ✨ **동적 크기 조정**: 메시지 길이에 맞춰 캡처 영역이 자동으로 크기 조정
- 📄 **PDF 인쇄 기능**: "활동 결과 인쇄하기"가 이제 A4 최적화 PDF 파일을 생성
- 🎯 **똑똑한 포맷**: 이미지 저장과 PDF 인쇄가 동일한 결과물을 다른 포맷으로 제공
- 🔄 **진행 상태 표시**: 파일 생성 중 진행 상황을 시각적으로 표시
- 📊 **글자 수 카운터**: 실시간 텍스트 길이 표시 및 색상 피드백
- ⚡ **스마트 스케일링**: A4 용지에 완벽하게 맞는 비율 자동 조정

### 🎯 **v2.4 기능 (유지)**
- ✨ **자동 저장**: 10초마다 작성 내용 자동 저장
- 🎯 **접근성 개선**: ARIA 레이블, 키보드 네비게이션 지원
- 💬 **피드백 시스템**: 실시간 토스트 메시지
- 📝 **자유 입력**: 엽서 메시지를 자유롭게 작성 가능
- 🖨️ **인쇄 최적화**: 웹과 인쇄용 스타일 분리

## 🎓 교육적 가치

- **정서 표현력 향상**: 다양한 매체(텍스트, 이미지, 이모지)를 통한 감정 표현
- **가족 관계 탐색**: 가족과의 소통과 관계 개선 동기 부여
- **성찰 능력 배양**: 메타인지적 감정 정리 과정
- **디지털 리터러시**: 웹 기반 도구 활용 능력 향상

## 💻 기술 스택

- **Frontend**: HTML5, CSS3 (Tailwind CSS), Vanilla JavaScript
- **Libraries**: 
  - `html2canvas`: 결과물 이미지 저장
  - `jsPDF`: PDF 생성
  - `Noto Sans KR`: 한글 폰트 최적화
- **Features**: 
  - 반응형 디자인 (모바일/태블릿 지원)
  - LocalStorage 활용 자동 저장
  - 실시간 미리보기
  - 동적 크기 조정
  - A4 최적화 PDF 생성
  - 2페이지 자동 분할 시스템

## 🎯 사용 방법

### **학생용**
1. **웹사이트 접속**: [GitHub Pages 링크](https://plusiam.github.io/family-happy-moments-v2/)
2. **활동 순서대로 진행**: 도입 → 활동1 → 활동2 → 마무리
3. **결과물 저장**: 
   - **🖼️ 이미지로 저장**: 메시지 길이에 맞춘 PNG 파일
   - **📄 활동 결과 인쇄하기**: A4 최적화 PDF 파일

### **교사용**
1. **수업 준비**: '선생님용 학습지 PDF 다운로드' 버튼 클릭 → **2페이지 완전 학습지** 다운로드
2. **학급 활용**: 개별 또는 모둠 활동으로 진행
3. **결과 수집**: 학생들의 PDF 파일로 포트폴리오 구성

## 💡 **v2.6의 혁신적 특징**

### **📄 완벽한 선생님용 학습지**
- **1페이지**: 도입 + 활동1 (엽서 만들기)
- **2페이지**: 활동2 (공감 이름표) + 마무리 + 추가 작성 공간
- **A4 최적화**: 210mm × 297mm 정확한 크기로 생성
- **내용 잘림 방지**: 페이지별 내용량 완벽 조절

### **⚡ 사용자 경험 개선**
- 선생님용 학습지 버튼에 "(빈 워크시트 PDF, 2페이지)" 명시
- 실시간 진행 상태 표시: "선생님용 학습지 PDF 생성 중... (2페이지)"
- 완료 메시지: "선생님용 학습지 PDF(2페이지)가 다운로드되었습니다!"

### **🎯 기술적 혁신**
- 각 페이지별 독립적인 html2canvas 캡처
- jsPDF addPage() 기능으로 자동 페이지 추가
- 794×1123 픽셀 정확한 A4 비율 적용
- 페이지별 숨김/표시 자동 처리

## 📊 **학습지 구성 상세**

### **📄 1페이지 내용**
```
✉️ 우리 가족, 행복한 순간 그리기
학교/학년/반/이름 입력란

🟡 도입: 생각 열기
- 행복한 순간 여부 체크
- 기억에 남는 장면 작성란

🔵 활동1: 엽서로 표현해요  
- 그림/사진 붙이기 공간
- 엽서 메시지 작성란 (4줄)
```

### **📄 2페이지 내용**
```
🔴 활동2: 공감 이름표 만들기
- 가족 이름 입력란
- 듣고 싶은 말 (2줄)
- 하고 싶은 말 (2줄)

🟢 마무리: 감정 정리하기
- 기분 이모지 선택란
- 이유 작성란 (3줄)

✍️ 추가로 하고 싶은 말
- 자유 작성 공간 (3줄)
```
